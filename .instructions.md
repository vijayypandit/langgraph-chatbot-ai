# Langgraph Chatbot Project Instructions

## Purpose
This app is a chatbot application built with Langgraph. Users can ask any query in natural language, and the chatbot returns a response using the underlying Langgraph-based model.

## Startup
1. Activate the project virtual environment.
   - Windows PowerShell: `.\.venv\Scripts\Activate.ps1`
   - Windows Command Prompt: `\.venv\Scripts\activate.bat`
2. Run the app:
   - `streamlit run streamlit_frontend.py`

## Environment
- Python version: 3.14+
- Use the workspace virtual environment located at `./.venv/`.
- Install dependencies from `requirements.txt` or `pyproject.toml` before running.

## Key files
- `streamlit_frontend.py` — main entry point for the Streamlit app.
- `langgraph_backend.py` — backend logic and Langgraph integration.
- `main.py` — optional additional entrypoint or app logic.
- `ui_config.py` — UI configuration settings.

## Workflow rules
- Create or update the `README.md` whenever the app files are changed by you or an agent.
- Update the theory/explanation content whenever a file is saved.
- Keep the main entrypoint `streamlit_frontend.py` stable and only modify it when necessary.

## Notes
- This project uses Langgraph and Streamlit, so run the app only after activating the virtual environment.
- If `streamlit` is not found, ensure the venv activation completed successfully and that the venv has the dependencies installed.
